There are two python files:
- nerc.ipynb : this one is for the NERC task.
- sentiment_topic.ipynb : this one is for the sentiment analysis and topic analysis tasks.

Required datasets:
Training:
- /CONLL2003/train.txt
- GoogleNews-vectors-negative300.bin.gz
- big_data.csv
- sst-1 (downloads automatically in the code)

Testing:
- sentiment-topic-test.tsv
- NER-test.tsv
